Mean, Median, Standard Deviation, IQR, Outlier Detection, and Statistical Synthesis

5 Marks
📖 Theory Background

Descriptive Statistics — What, Why, How

Descriptive statistics convert raw data into meaningful summaries covering three aspects:

  • Central Tendency: Where is the "centre"? → Mean (arithmetic average), Median (middle value), Mode (most frequent)
  • Dispersion: How spread out is the data? → Range, Variance, Standard Deviation, IQR
  • Shape / Outliers: Symmetric or skewed? Any extreme values?

Mean $\bar{x}$ = Sum of all values ÷ $n$. Sensitive to extreme values (outliers pull it).

Median = Middle value when sorted. Robust against outliers. For even $n$: average of the $(n/2)$-th and $(n/2+1)$-th sorted values.

Sample Standard Deviation $s$ uses $n-1$ (Bessel's correction) to give an unbiased estimate of population SD from a sample.

Quartiles: $Q_1$ = 25th percentile (25% of data below), $Q_3$ = 75th percentile. IQR $= Q_3 - Q_1$ spans the middle 50%.

1.5×IQR Rule (Tukey Fences): Values below $Q_1 - 1.5\cdot\text{IQR}$ or above $Q_3 + 1.5\cdot\text{IQR}$ are flagged as potential outliers.

Core Formulas
$$\bar{x} = \frac{\sum x_i}{n} \qquad s = \sqrt{\frac{\sum(x_i-\bar{x})^2}{n-1}} \qquad \text{IQR} = Q_3 - Q_1$$ $$\text{Lower Fence} = Q_1 - 1.5\times\text{IQR} \qquad \text{Upper Fence} = Q_3 + 1.5\times\text{IQR}$$
🔧 Algorithm
⚙️ How to Compute Descriptive Statistics
1
Sort data ascendingWHAT: Arrange smallest to largest. WHY: All positional statistics (median, quartiles) require ordered data. HOW: Simply reorder the values.
2
Compute MeanWHAT: Arithmetic average. WHY: Most common measure of centre. HOW: $\bar{x} = \sum x_i / n$.
3
Find MedianWHAT: Middle value. WHY: Robust central measure, unaffected by extremes. HOW: For even $n$, average the two middle values.
4
Compute Q1 and Q3WHAT: Lower/upper quartiles. WHY: Required for IQR and fences. HOW: $Q_1$ position = $0.25(n+1)$; $Q_3$ position = $0.75(n+1)$. Interpolate if fractional.
5
Compute IQR and FencesWHAT: Spread of middle 50%; outlier boundaries. WHY: Identifies unusual observations. HOW: $\text{IQR}=Q_3-Q_1$; Fences: $Q_1 \pm 1.5\times\text{IQR}$.
6
Compute Sample SDWHAT: Average distance from mean. WHY: Quantifies variability. HOW: $(x_i-\bar{x})^2$ for each, sum all, divide by $n-1$, take square root.
7
Flag outliersWHAT: Extreme values beyond fences. WHY: Need investigation — data error or genuine anomaly. HOW: Check each value against Lower/Upper Fence.

📖 Theory — Naive Bayes

Naive Bayes Classifier

Naive Bayes is a probabilistic classifier based on Bayes' Theorem. "Naive" means we assume all features (words) are conditionally independent given the class. Despite this simplification, it performs remarkably well for text classification.

Core Idea: For a new document, compute a score for each class proportional to the product of the prior probability and the likelihoods of each word. Assign the class with the highest score.

Why Laplace Smoothing? Without smoothing, if a word never appears in training for a class, its likelihood = 0, zeroing out the entire product. Adding pseudocount $\alpha=1$ prevents this.

  • $P(w|C) = \dfrac{\text{count}(w,C) + \alpha}{\text{total words in }C + \alpha\times|V|}$
  • $|V|$ = vocabulary size (count of unique words across ALL training documents)
Naive Bayes Formula
$$P(C|d) \propto P(C) \times \prod_{w \in d} P(w|C) \qquad P(w|C) = \frac{\text{count}(w,C)+1}{\text{total}_C + |V|}$$
🔧 Algorithm
⚙️ Naive Bayes Steps
1
Compute class priors$P(C)$ = count of documents in class C / total documents.
2
Build vocabulary $|V|$List all unique words across the entire training set and count them.
3
Count word frequencies per classFor each class, count occurrences of every word. Sum = total words in that class.
4
Apply Laplace smoothing for test words$P(w|C) = (\text{count}(w,C)+1)/(\text{total}_C + |V|)$.
5
Compute posterior scoresFor each class: Prior × product of word likelihoods for all test words.
6
ClassifyAssign the class with the highest posterior score.

📖 Theory — Probability Mass Function

Probability Mass Function (PMF)

A PMF $p(x)$ assigns a probability to each discrete value of a random variable $X$. Two axioms must hold:

  • Non-negativity: $p(x) \geq 0$ for all $x$
  • Completeness: $\sum_{\text{all }x} p(x) = 1$

Using these axioms to find unknown constants (like $k$) is the foundation of PMF problems.

Key probability rules:

  • $P(X < b) = \sum_{x < b} p(x)$  (excludes $x=b$)
  • $P(X \geq a) = 1 - P(X < a)$  (complement rule)
  • $P(a < X < b) = \sum_{a < x < b} p(x)$  (strict inequalities exclude endpoints)
PMF Key Conditions
$$\sum_{\text{all }x} p(x) = 1 \qquad p(x) \geq 0 \qquad P(X \geq a) = 1 - P(X < a)$$

📖 Theory — Poisson Distribution

The Poisson Distribution

The Poisson distribution models the number of rare, random events occurring in a fixed interval, given a constant average rate $\lambda$. Key conditions:

  • Events occur independently of each other
  • Average rate $\lambda$ is constant
  • Two events cannot happen at exactly the same instant

Poisson as approximation to Binomial: When $n$ is large (e.g., ≥30) and $p$ is small (e.g., ≤0.05), Binomial$(n,p)$ is well approximated by Poisson($\lambda = np$). Here: $n=10$ blades, $p=0.002$, $np=0.02$.

$e^{-\lambda}$ values: $e^{-0.02} \approx 0.9802$, $e^{-0.5} \approx 0.6065$, $e^{-3} \approx 0.0498$

Poisson PMF
$$P(X=k) = \frac{e^{-\lambda}\lambda^k}{k!} \quad k=0,1,2,\ldots \qquad \lambda = np$$

📖 Theory — Joint Continuous Distributions

Joint PDF: Complete Framework

A joint PDF $f(x,y)$ describes the simultaneous distribution of two continuous random variables. It must satisfy:

  • $f(x,y) \geq 0$ for all $(x,y)$
  • $\displaystyle\iint f(x,y)\,dx\,dy = 1$

Marginal PDFs: $f_X(x) = \int f(x,y)\,dy$ (integrate out $y$); $f_Y(y) = \int f(x,y)\,dx$ (integrate out $x$).

Independence test: $X \perp Y \iff f(x,y) = f_X(x)\cdot f_Y(y)$ for all $(x,y)$. A separable joint PDF always implies independence.

Joint probability: $P(a\leq X\leq b,\, c\leq Y\leq d) = \int_a^b\int_c^d f(x,y)\,dy\,dx$

Expectations: $E[X]=\int x f_X(x)\,dx$; $E[XY]=\iint xy f(x,y)\,dx\,dy$. If independent: $E[XY]=E[X]E[Y]$.

Joint PDF Formulas
$$f_X(x)=\int f(x,y)\,dy \qquad f_Y(y)=\int f(x,y)\,dx$$ $$E[X]=\int x\,f_X(x)\,dx \qquad E[XY]=\iint xy\,f(x,y)\,dx\,dy$$
💡 Practice Examples
1
$f(x,y)=2$ for $0\leq x\leq y\leq 1$. Verify validity.

$\int_0^1\int_0^y 2\,dx\,dy=\int_0^1 2y\,dy=[y^2]_0^1=1$ ✔ Valid!

2
$f(x,y)=x+y$, $0\leq x,y\leq1$. Find $f_X(x)$ and $E[X]$.

$f_X(x)=\int_0^1(x+y)\,dy=x+1/2$ | $E[X]=\int_0^1 x(x+1/2)\,dx=1/3+1/4=7/12$

3
$f(x,y)=4xy$, $0\leq x,y\leq1$. Are X and Y independent?

$f_X(x)=2x$ | $f_Y(y)=2y$ | $f_X\cdot f_Y=4xy=f(x,y)$ → YES, independent!

✅ Complete Worked Solution — $f(x,y)=6x^2y$, $0\leq x,y\leq1$

Part (i) — Verify Valid Joint PDF

$f(x,y)=6x^2y\geq0$ for $x,y\in[0,1]$ ✔ (non-negativity satisfied)

The function separates: $6x^2y = 6\cdot x^2\cdot y$

$$\int_0^1\int_0^1 6x^2y\,dx\,dy = 6\cdot\int_0^1 x^2\,dx\cdot\int_0^1 y\,dy = 6\cdot\frac{1}{3}\cdot\frac{1}{2} = 6\cdot\frac{1}{6} = \mathbf{1} \checkmark$$

Part (ii) — Marginal Densities

$f_X(x)$:

$$f_X(x) = \int_0^1 6x^2y\,dy = 6x^2\left[\frac{y^2}{2}\right]_0^1 = 6x^2\cdot\frac{1}{2} = \mathbf{3x^2}, \quad 0\leq x\leq1$$

$f_Y(y)$:

$$f_Y(y) = \int_0^1 6x^2y\,dx = 6y\left[\frac{x^3}{3}\right]_0^1 = 6y\cdot\frac{1}{3} = \mathbf{2y}, \quad 0\leq y\leq1$$

Part (iii) — $P(X < 0.5,\, Y > 0.5)$

$$P = \int_0^{0.5}\int_{0.5}^1 6x^2y\,dy\,dx$$

Inner integral: $\displaystyle\int_{0.5}^1 6x^2y\,dy = 6x^2\left[\frac{y^2}{2}\right]_{0.5}^1 = 6x^2\left(\frac{1}{2}-\frac{1}{8}\right) = 6x^2\cdot\frac{3}{8} = \frac{18x^2}{8}$

Outer integral: $\displaystyle\int_0^{0.5}\frac{18x^2}{8}\,dx = \frac{18}{8}\cdot\frac{(0.5)^3}{3} = \frac{18}{8}\cdot\frac{1}{24} = \frac{18}{192} = \frac{3}{32} = \mathbf{0.09375}$

Part (iv) — Independence of X and Y

$$f_X(x)\cdot f_Y(y) = 3x^2\cdot 2y = 6x^2y = f(x,y) \checkmark$$

✅
X and Y are INDEPENDENT! The joint PDF factors exactly into the product of the marginals. This follows from $f(x,y)$ being separable: $6x^2y = (3x^2)(2y) = f_X(x)f_Y(y)$.

Part (v) — $E[X]$, $E[Y]$, $E[XY]$

$$E[X] = \int_0^1 x\cdot3x^2\,dx = 3\int_0^1 x^3\,dx = 3\cdot\frac{1}{4} = \mathbf{\frac{3}{4} = 0.75}$$

$$E[Y] = \int_0^1 y\cdot2y\,dy = 2\int_0^1 y^2\,dy = 2\cdot\frac{1}{3} = \mathbf{\frac{2}{3} \approx 0.667}$$

Since X and Y are independent: $E[XY]=E[X]\cdot E[Y]$

$$E[XY] = \frac{3}{4}\times\frac{2}{3} = \frac{1}{2} = \mathbf{0.5}$$

Verification: $\iint xy\cdot6x^2y\,dx\,dy = 6\int_0^1 x^3\,dx\cdot\int_0^1 y^2\,dy = 6\cdot\frac{1}{4}\cdot\frac{1}{3} = \frac{1}{2}$ ✔

📖 Theory — Normal Distribution

The Normal (Gaussian) Distribution

$X\sim N(\mu,\sigma^2)$: symmetric bell-shaped curve centred at $\mu$ with spread $\sigma$.

  • 68-95-99.7 Rule: 68% within $\mu\pm\sigma$; 95% within $\mu\pm2\sigma$; 99.7% within $\mu\pm3\sigma$
  • Standardisation: $Z = (X-\mu)/\sigma$ converts any normal to Standard Normal $N(0,1)$
  • Use Z-table $\Phi(z) = P(Z\leq z)$ to find probabilities
  • Inverse: For cutoff $x$ such that $P(X\leq x)=p$: $x=\mu+z_p\cdot\sigma$ where $\Phi(z_p)=p$

Key Z-table values: $\Phi(1.2)=0.8849$; $\Phi(-1.6)=0.0548$; $\Phi(2.25)=0.9878$; $\Phi(1.645)=0.95$; $\Phi(2.0)=0.9772$

Normal Distribution Key Formulas
$$Z = \frac{X-\mu}{\sigma} \qquad P(a\leq X\leq b) = \Phi(z_b)-\Phi(z_a) \qquad x = \mu+z\sigma$$
💡 Practice Examples

📖 Theory — Inclusion-Exclusion Principle

Inclusion-Exclusion & Event Algebra

For three events $A$, $B$, $C$:

$$P(A\cup B\cup C) = P(A)+P(B)+P(C) - P(A\cap B) - P(B\cap C) - P(C\cap A) + P(A\cap B\cap C)$$

Complement Rule: $P(\bar{A}\cap\bar{B}\cap\bar{C}) = 1 - P(A\cup B\cup C)$ (probability all fail = 1 minus probability at least one occurs)

Exactly two events: $P(\text{exactly 2}) = [P(AB)+P(BC)+P(CA)] - 3P(ABC)$

Exactly one event: $P(\text{exactly 1}) = P(\geq1) - P(\text{exactly 2}) - P(\text{all 3})$

Sanity check: $P(\text{none}) + P(\text{exactly 1}) + P(\text{exactly 2}) + P(\text{all 3}) = 1$

Event Count Formulas
$$P(\geq1) = 1 - P(\text{none}) \qquad P(\text{exactly 2}) = \sum P(A_iA_j) - 3P(ABC)$$ $$P(\text{exactly 1}) = P(\geq1) - P(\text{exactly 2}) - P(\text{all 3})$$

The Five-Number Summary & Box Plots

The five-number summary provides a complete distributional picture using just five statistics:

  • Minimum: Smallest observed value

  • Median (Q2): 50th percentile — the middle of the dataset

📖 Theory — Bayes' Theorem

Bayes' Theorem: Updating Beliefs with Evidence

Bayes' Theorem reverses conditional probability: given $P(\text{effect}|\text{cause})$, find $P(\text{cause}|\text{effect})$.

Setup: Mutually exclusive, exhaustive hypotheses $H_1,\ldots,H_k$ with prior probabilities. We observe evidence $E$.

Total Probability Theorem: $P(E) = \sum_i P(E|H_i)\cdot P(H_i)$

Bayes' Formula:

$$P(H_i|E) = \frac{P(E|H_i)\cdot P(H_i)}{\sum_j P(E|H_j)\cdot P(H_j)}$$

Intuition: Posterior $\propto$ Prior $\times$ Likelihood. Evidence re-weights our belief about which hypothesis is true.

Bayes' Theorem
$$P(H_i|E) = \frac{P(E|H_i)\cdot P(H_i)}{P(E)} \qquad P(E)=\sum_j P(E|H_j)\cdot P(H_j)$$

📖 Theory — Joint Discrete Distributions

Joint PMF for Discrete Random Variables

A joint PMF $f(x,y)=P(X=x,Y=y)$ satisfies: $f(x,y)\geq0$ and $\sum_x\sum_y f(x,y)=1$.

  • Marginal PMF: $f_X(x)=\sum_y f(x,y)$  (sum over all $y$); $f_Y(y)=\sum_x f(x,y)$
  • Independence: $X\perp Y \iff f(x,y)=f_X(x)\cdot f_Y(y)$ for ALL $(x,y)$ pairs
  • Conditional probability: $P(A|B)=P(A\cap B)/P(B)$

To check independence, a single counterexample where $f(x,y)\neq f_X(x)f_Y(y)$ is enough to prove dependence.

Joint PMF Relations
$$\sum_x\sum_y f(x,y)=1 \qquad f_X(x)=\sum_y f(x,y) \qquad P(A|B)=\frac{P(A\cap B)}{P(B)}$$

📖 Theory — CDF and PDF Relationship

CDF and PDF: Two Sides of the Same Coin

The CDF $F(x) = P(X\leq x)$ gives cumulative probability up to $x$.

Valid CDF Properties:

  • Non-decreasing function of $x$
  • $\lim_{x\to-\infty}F(x)=0$ and $\lim_{x\to+\infty}F(x)=1$
  • Right-continuous

PDF from CDF: $f(x) = F'(x) = \dfrac{d}{dx}F(x)$

Probability from CDF: $P(a\leq X\leq b)=F(b)-F(a)$

Finding k: Use boundary condition $F(\text{upper bound of support})=1$.

CDF ⇆ PDF Relations
$$f(x)=F'(x) \qquad F(x)=\int_{-\infty}^x f(t)\,dt \qquad P(a\leq X\leq b)=F(b)-F(a)$$

Poisson for Counting Rare Events

$X\sim\text{Poisson}(\lambda)$: number of events in a fixed period with average rate $\lambda$.

Here: error rate = 1.2 per 100 cheques; teller processes 250 cheques per day.

$\lambda = (1.2/100)\times250 = 3$ errors per day.

Useful values with $\lambda=3$: $e^{-3}\approx0.04979$

$P(X=k)=e^{-3}\cdot3^k/k!$: $P(0)=0.0498$; $P(1)=0.1494$; $P(2)=0.2240$; $P(3)=0.2240$; $P(4)=0.1680$

Inclusion-exclusion to find defective and defect-free disc counts

2 Marks
📖 Theory — Inclusion-Exclusion for Counting

Inclusion-Exclusion Principle

For three sets $W$, $C$, $S$:

$$|W\cup C\cup S| = |W|+|C|+|S| - |W\cap C| - |W\cap S| - |C\cap S| + |W\cap C\cap S|$$

Logic: (1) Add individual sets (overcounts intersections). (2) Subtract pairwise intersections (now triple counted too little). (3) Add back triple intersection once.

Defect-free count: $N - |W\cup C\cup S|$

💡 Practice Example
1
Class of 50: Cricket=20, Football=15, Basketball=10. C∩F=5, C∩B=4, F∩B=3, all three=2. How many play at least one?

$|C\cup F\cup B|=20+15+10-5-4-3+2=35$ | Non-players = 50-35 = 15

Part (i) — P(At Least One Defect)

$$|W\cup C\cup S| = 1200+850+1500-300-400-250+100 = 3550-950+100 = \mathbf{2700}$$

$$P(\text{at least one defect}) = \frac{2700}{10000} = \mathbf{0.27 = 27\%}$$

Part (ii) — Number of Defect-Free Discs

$$\text{Defect-free} = N - |W\cup C\cup S| = 10000 - 2700 = \mathbf{7300 \text{ discs}}$$

$$P(\text{defect-free}) = \frac{7300}{10000} = 73\%$$
