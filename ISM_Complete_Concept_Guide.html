Descriptive Statistics

Central Tendency, Variability, and Sampling — CS1 Module

Five-Number Summary & Box Plot

The five-number summary gives a complete picture of any dataset's spread without assuming any particular distribution. It consists of five key values: Minimum, Q1, Median, Q3, Maximum.

Position Formula
Q1 position = \( \frac{1}{4}(n+1) \)
Q2 (Median) = \( \frac{1}{2}(n+1) \)
Q3 position = \( \frac{3}{4}(n+1) \)
IQR & Skewness
IQR = Q3 − Q1
Lower Fence = Q1 − 1.5 × IQR
Upper Fence = Q3 + 1.5 × IQR
Skewness (Rough Guide)
Mean > Median → Positive (Right) Skew
Mean < Median → Negative (Left) Skew
Mean ≈ Median → Symmetric

🔑 Exam Tips

  • Always sort the data FIRST before finding any quartile
  • If position is decimal (e.g., 3.75), interpolate: value at 3 + 0.75×(value at 4 − value at 3)
  • For even n, median = average of n/2 th and (n/2 + 1)th values
✏️ Practice Example
Problem: The daily sales (in ₹ thousands) at a shop for 9 days are:
12, 7, 25, 15, 9, 22, 31, 18, 14
Find the Five-Number Summary and detect any outliers.
1
Sort the data
Arrange in ascending order: 7, 9, 12, 14, 15, 18, 22, 25, 31 (n = 9)
2
Find Minimum, Maximum, Median
Min = 7, Max = 31
Median position = \( \frac{9+1}{2} = 5^{th} \) value = 15
3
Find Q1 and Q3
Q1 position = \( \frac{1}{4}(9+1) = 2.5^{th} \) → \( 9 + 0.5(12-9) = 10.5 \)
Q3 position = \( \frac{3}{4}(9+1) = 7.5^{th} \) → \( 22 + 0.5(25-22) = 23.5 \)
4
Compute IQR and Fences
IQR = Q3 − Q1 = 23.5 − 10.5 = 13
Lower Fence = 10.5 − 1.5(13) = 10.5 − 19.5 = −9
Upper Fence = 23.5 + 1.5(13) = 23.5 + 19.5 = 43
All data values lie between −9 and 43, so no outliers.
Final Answer
Min = 7, Q1 = 10.5, Median = 15, Q3 = 23.5, Max = 31 | IQR = 13 | No outliers detected.
Mean and Standard Deviation (Step-by-Step Tabular Method)

Standard deviation measures how far data points are spread from the mean. Always show the full table in exams — each step earns marks.

Sample Mean
\[ \bar{x} = \frac{\sum x_i}{n} \]
Sample Variance
\[ s^2 = \frac{\sum (x_i - \bar{x})^2}{n - 1} \]
Sample Std Deviation
\[ s = \sqrt{s^2} = \sqrt{\frac{\sum (x_i - \bar{x})^2}{n - 1}} \]

🔑 Why n−1 (not n)?

  • We use n−1 (Bessel's correction) for sample standard deviation
  • It makes the estimate unbiased — corrects for the fact that a sample underestimates population spread
  • For population standard deviation (σ), use N in denominator
✏️ Practice Example
Problem: Students scored the following marks in a quiz: 45, 52, 38, 61, 55. Find the mean and standard deviation.
1
Calculate the Mean
\[ \bar{x} = \frac{45 + 52 + 38 + 61 + 55}{5} = \frac{251}{5} = 50.2 \]
2
Build the Deviation Table
\(x_i\)\(x_i - \bar{x}\)\((x_i - \bar{x})^2\)
45−5.227.04
521.83.24
38−12.2148.84
6110.8116.64
554.823.04
Sum≈ 0318.80
3
Calculate Variance and Standard Deviation
\[ s^2 = \frac{318.80}{5 - 1} = \frac{318.80}{4} = 79.70 \]
\[ s = \sqrt{79.70} \approx 8.93 \]
Final Answer
Mean = 50.2, Standard Deviation ≈ 8.93 marks
IQR Rule and Outlier Detection

Any data point falling below the Lower Fence or above the Upper Fence is considered a potential outlier. This is the Tukey Fence method (1.5 × IQR rule).

Tukey Fences
Lower Fence = Q1 − 1.5 × IQR
Upper Fence = Q3 + 1.5 × IQR
Outlier if: x < LF or x > UF
Central Limit Theorem (CLT)

The CLT states: if we take many samples of size n from ANY population with mean μ and standard deviation σ, then the distribution of sample means \(\bar{X}\) will be approximately Normal as n increases.

Standard Error (SE)
\[ SE = \frac{\sigma}{\sqrt{n}} \]
(Std dev of the sampling distribution)
Z-score for Sample Mean
\[ Z = \frac{\bar{X} - \mu}{SE} = \frac{\bar{X} - \mu}{\sigma / \sqrt{n}} \]
✏️ Practice Example
Problem: The average height of adult males is 70 inches with standard deviation 3 inches. A random sample of 49 males is selected. Find P(69 < X̄ < 71).
1
Compute Standard Error
\[ SE = \frac{\sigma}{\sqrt{n}} = \frac{3}{\sqrt{49}} = \frac{3}{7} = 0.4286 \]
2
Convert to Z-scores
\[ Z_1 = \frac{69 - 70}{0.4286} = -2.333 \qquad Z_2 = \frac{71 - 70}{0.4286} = +2.333 \]
3
Look up Z-table and compute probability
\[ P(-2.333 < Z < 2.333) = P(Z < 2.333) - P(Z < -2.333) \] \[ = 0.9902 - 0.0098 = \mathbf{0.9804} \]
Final Answer
P(69 < X̄ < 71) = 0.9804 → 98.04% chance the sample mean falls between 69 and 71 inches.
Confidence Intervals & Required Sample Size

A confidence interval gives a range in which the true population parameter likely falls. The wider the interval, the more confident we are. The required sample size ensures the margin of error stays within a specified limit.

Margin of Error
\[ E = z_{\alpha/2} \cdot \frac{\sigma}{\sqrt{n}} \]
Required Sample Size
\[ n = \left(\frac{z_{\alpha/2} \cdot \sigma}{E}\right)^2 \]
Always round UP to next integer!
Common Z-values
90% CI → z = 1.645
95% CI → z = 1.960
99% CI → z = 2.576
✏️ Practice Example (From Exam Paper)
Problem: A sample of 49 workers spent an average of 45 minutes in the canteen with SD = 14 minutes. How large a sample is needed for a margin of error ≤ 2.5 minutes at 99% confidence?
1
Identify the required Z-value for 99% confidence
For 99% CI: z = 2.576
2
Apply the sample size formula
\[ n = \left(\frac{z \cdot \sigma}{E}\right)^2 = \left(\frac{2.576 \times 14}{2.5}\right)^2 \] \[ = \left(\frac{36.064}{2.5}\right)^2 = (14.426)^2 = 208.11 \]
3
Round UP to ensure adequate precision
n = ⌈208.11⌉ = 209 workers
Final Answer
Required sample size = 209 workers

Probability Theory

Set Theory, Conditional Probability, Total Probability, Bayes' Theorem & Naïve Bayes

Set Theory & Basic Probability Rules

Events are sets. Probability follows the same rules as set theory. These identities are repeatedly tested in exams.

Addition Rule
\[ P(A \cup B) = P(A) + P(B) - P(A \cap B) \]
Complement
\[ P(A') = 1 - P(A) \]
\[ P(A \cap B') = P(A) - P(A \cap B) \]
De Morgan's & Decomposition
\[ P(A \cup B)' = P(A' \cap B') \]
\[ P(A) = P(A - B) + P(A \cap B) \]
✏️ Practice Example (From Exam Paper)
Problem: In a company, P(M ∪ Q) = 0.40 (machine or quality inspectors), P(M − Q) = 0.15 (only machine operators), P(Q − M) = 0.10 (only quality inspectors). Find P(M ∩ Q), P(M), P(Q).
1
Use decomposition: P(M ∪ Q) = P(M−Q) + P(Q−M) + P(M ∩ Q)
0.40 = 0.15 + 0.10 + P(M ∩ Q)
\[ P(M \cap Q) = 0.40 - 0.25 = \mathbf{0.15} \]
2
Find P(M) = P(M−Q) + P(M ∩ Q)
P(M) = 0.15 + 0.15 = 0.30
3
Find P(Q) = P(Q−M) + P(M ∩ Q)
P(Q) = 0.10 + 0.15 = 0.25
Final Answer
P(M ∩ Q) = 0.15, P(M) = 0.30, P(Q) = 0.25
Conditional Probability

P(A|B) is the probability of A occurring given that B has already occurred. It "narrows" the sample space to only outcomes where B is true.

Definition
\[ P(A|B) = \frac{P(A \cap B)}{P(B)} \]
(requires P(B) > 0)
Multiplication Rule
\[ P(A \cap B) = P(A|B) \cdot P(B) \] \[ = P(B|A) \cdot P(A) \]
Independence
If A and B are independent:
\[ P(A \cap B) = P(A) \cdot P(B) \]
\[ P(A|B) = P(A) \]
✏️ Practice Example (From Exam Paper)
Problem: P(Q) = 0.35 (keeps quiet), P(S|Q) = 0.75 (calls in sick given keeps quiet), P(Q|S) = 0.40 (keeps quiet given calls in sick). Find: P(Q ∩ S), P(S), P(Q ∪ S), P(Q'|S), P(Q' ∩ S').
1
Find P(Q ∩ S) using Multiplication Rule
\[ P(Q \cap S) = P(S|Q) \times P(Q) = 0.75 \times 0.35 = \mathbf{0.2625} \]
2
Find P(S) by rearranging P(Q|S)
\[ P(S) = \frac{P(Q \cap S)}{P(Q|S)} = \frac{0.2625}{0.40} = \mathbf{0.65625} \]
3
Find P(Q ∪ S)
\[ P(Q \cup S) = 0.35 + 0.65625 - 0.2625 = \mathbf{0.74375} \]
4
Remaining parts
P(Q'|S) = 1 − P(Q|S) = 1 − 0.40 = 0.60
P(Q' ∩ S') = 1 − P(Q ∪ S) = 1 − 0.74375 = 0.25625
P(Q ∩ S') = P(Q) − P(Q ∩ S) = 0.35 − 0.2625 = 0.0875
Law of Total Probability

If an event can happen via multiple mutually exclusive paths (the "partition" of the sample space), sum the probability through each path. Think of it as adding all the routes to a destination.

Total Probability
\[ P(B) = \sum_{i} P(B | A_i) \cdot P(A_i) \]
where \(A_i\) are mutually exclusive and exhaustive.
✏️ Practice Example (From Exam Paper)
Problem: 70% of messages go to Account 1 (1% spam), 20% to Account 2 (2% spam), 10% to Account 3 (5% spam). Find P(Spam).
1
Identify partitions (accounts) and their spam rates
P(A₁) = 0.70, P(Spam|A₁) = 0.01
P(A₂) = 0.20, P(Spam|A₂) = 0.02
P(A₃) = 0.10, P(Spam|A₃) = 0.05
2
Apply Total Probability formula
\[ P(\text{Spam}) = (0.01)(0.70) + (0.02)(0.20) + (0.05)(0.10) \] \[ = 0.007 + 0.004 + 0.005 = \mathbf{0.016} \]
Final Answer
P(Spam) = 0.016 = 1.6%
Bayes' Theorem

Bayes' theorem lets us reverse conditional probability. Given that we observed evidence (result), what's the probability the underlying cause is true? It combines prior knowledge with new evidence.

Bayes' Theorem
\[ P(A|B) = \frac{P(B|A) \cdot P(A)}{P(B)} \]
\[ = \frac{P(B|A) \cdot P(A)}{\sum_i P(B|A_i)P(A_i)} \]
Terminology
P(A): Prior probability of A
P(B|A): Likelihood
P(B): Total/marginal probability
P(A|B): Posterior probability

🔑 3-Step Recipe for Bayes' Problems

  • Step 1: Identify and list all given probabilities
  • Step 2: Calculate P(Evidence) using Total Probability
  • Step 3: Plug into Bayes formula: P(Cause|Evidence) = P(E|Cause) × P(Cause) / P(Evidence)
✏️ Practice Example (From Exam Paper)
Problem: 1 in 1000 adults has a rare disease. A positive test occurs 99% of the time if sick, and 2% of the time if healthy. If a random person tests positive, what is P(disease)?
1
List all given probabilities
P(D) = 0.001, P(D') = 0.999
P(+|D) = 0.99 (sensitivity), P(+|D') = 0.02 (false positive rate)
2
Compute P(+) using Total Probability
\[ P(+) = P(+|D)P(D) + P(+|D')P(D') \] \[ = (0.99)(0.001) + (0.02)(0.999) = 0.00099 + 0.01998 = 0.02097 \]
3
Apply Bayes' Theorem
\[ P(D|+) = \frac{P(+|D) \cdot P(D)}{P(+)} = \frac{0.00099}{0.02097} \approx \mathbf{0.0472} \]
Final Answer
Even with a positive test, P(disease) ≈ 4.72%. This counter-intuitive result is called the "Base Rate Fallacy"!
Naïve Bayes Classifier

Naïve Bayes is a machine learning classification algorithm based on Bayes' theorem with an independence assumption between features. Despite being "naïve", it performs excellently in spam filtering and text classification.

Classification Rule
\[ \hat{y} = \arg\max_{c} P(c) \prod_{i} P(w_i | c) \]
Choose class with highest posterior score.
Laplace Smoothing
\[ P(w|c) = \frac{\text{count}(w, c) + 1}{\text{total words in c} + |V|} \]
|V| = vocabulary size

🔑 Step-by-Step Algorithm

  • Step 1: Count how many training examples are in each class → compute P(class)
  • Step 2: Count word frequencies within each class
  • Step 3: Apply Laplace smoothing to compute P(word|class)
  • Step 4: For the test message, multiply: P(class) × product of P(each word|class)
  • Step 5: Assign the class with the HIGHER score
✏️ Practice Example (From Exam Paper)
Problem: Dataset: "Win money" → Spam, "Free prize" → Spam, "See you soon" → Ham, "Call me now" → Ham. Classify the new message "Win free money" as Spam or Ham.
1
Prior Probabilities
P(Spam) = 2/4 = 0.5, P(Ham) = 2/4 = 0.5
2
Build vocabulary and count words
Vocabulary: {win, money, free, prize, see, you, soon, call, me, now} → |V| = 10
Spam words: win(1), money(1), free(1), prize(1) → total = 4
Ham words: see(1), you(1), soon(1), call(1), me(1), now(1) → total = 6
3
Apply Laplace smoothing for test words: "win", "free", "money"
P(win|Spam) = (1+1)/(4+10) = 2/14; P(free|Spam) = 2/14; P(money|Spam) = 2/14
P(win|Ham) = (0+1)/(6+10) = 1/16; P(free|Ham) = 1/16; P(money|Ham) = 1/16
4
Calculate scores
Score(Spam) ∝ 0.5 × (2/14)³ = 0.5 × 8/2744 ≈ 0.001457
Score(Ham) ∝ 0.5 × (1/16)³ = 0.5 × 1/4096 ≈ 0.0001221
Final Answer
Score(Spam) > Score(Ham) → Message classified as SPAM ✓

Probability Distributions

Discrete & Continuous Random Variables, Binomial, Poisson, Normal — CS5 & CS6

Discrete Random Variables — PMF, E(X), Var(X)

A discrete random variable takes countable values (0, 1, 2, ...). Its Probability Mass Function (PMF) assigns probabilities to each value. Key requirement: all probabilities sum to exactly 1.

Valid PMF
\[ \sum_x P(X=x) = 1 \]
\[ P(X=x) \geq 0 \text{ for all } x \]
Expectation (Mean)
\[ E(X) = \sum_x x \cdot P(X=x) \]
Variance
\[ \text{Var}(X) = E(X^2) - [E(X)]^2 \]
\[ E(X^2) = \sum_x x^2 \cdot P(X=x) \]
Linear Transform
\[ E(aX + b) = aE(X) + b \]
\[ \text{Var}(aX + b) = a^2 \text{Var}(X) \]
✏️ Practice Example (From Exam Paper)
Problem: P(X=x) = kx² for x = −1, 0, 1 and 0 otherwise. Find: (i) k, (ii) E[X], (iii) Var(X).
1
Find k using ΣP(X=x) = 1
k(−1)² + k(0)² + k(1)² = 1
k(1) + k(0) + k(1) = 1 → 2k = 1 → k = 1/2
PMF: P(−1) = 1/2, P(0) = 0, P(1) = 1/2
2
Find E[X]
\[ E(X) = (-1)(1/2) + (0)(0) + (1)(1/2) = -0.5 + 0 + 0.5 = \mathbf{0} \]
3
Find Var(X) = E(X²) − [E(X)]²
E(X²) = (1)(1/2) + (0)(0) + (1)(1/2) = 1
\[ \text{Var}(X) = 1 - 0^2 = \mathbf{1} \]
Final Answer
k = 1/2, E(X) = 0, Var(X) = 1
Joint PMF and Marginal Distributions

A Joint PMF describes the probability of two random variables taking specific values simultaneously. Marginal PMFs are obtained by summing out one variable.

Marginal PMF of X
\[ P_X(x) = \sum_y P(X=x, Y=y) \]
(Sum down each column)
Marginal PMF of Y
\[ P_Y(y) = \sum_x P(X=x, Y=y) \]
(Sum across each row)
Conditional PMF
\[ P(X=x | Y=y) = \frac{P(X=x, Y=y)}{P_Y(y)} \]
✏️ Practice Example
Problem: From a joint PMF table, find the marginal PMF of X and the conditional P(X=1|Y=2):
Joint PMF: P(0,2)=0.01, P(1,2)=0.06, P(2,2)=0.11, P(3,2)=0.04. P_Y(2) = ?
1
Find P_Y(2) by summing column Y=2
P_Y(2) = 0.01 + 0.06 + 0.11 + 0.04 = 0.22
2
Apply Conditional PMF formula
\[ P(X=1|Y=2) = \frac{P(X=1, Y=2)}{P_Y(2)} = \frac{0.06}{0.22} \approx \mathbf{0.2727} \]
Final Answer
P(X=1|Y=2) ≈ 0.2727 (27.27%)
Hypergeometric Distribution

Used when drawing samples without replacement from a population with two categories (e.g., defective/non-defective). Unlike Binomial, trials are NOT independent.

Hypergeometric PMF
\[ P(X=x) = \frac{\binom{K}{x}\binom{N-K}{n-x}}{\binom{N}{n}} \]
N=population, K=defective, n=sample size
Mean & Variance
\[ E(X) = n \cdot \frac{K}{N} \]
\[ \text{Var}(X) = n\frac{K}{N}\frac{N-K}{N}\frac{N-n}{N-1} \]
✏️ Practice Example (From Exam Paper)
Problem: From 10 items (3 defective), 4 are drawn without replacement. Find P(X=2) and E(X).
1
Identify parameters: N=10, K=3, n=4
Total denominator: \(\binom{10}{4} = 210\)
2
Calculate P(X=2)
\[ P(X=2) = \frac{\binom{3}{2}\binom{7}{2}}{\binom{10}{4}} = \frac{3 \times 21}{210} = \frac{63}{210} = \mathbf{\frac{3}{10}} \]
3
Calculate E(X) using the shortcut
\[ E(X) = n \cdot \frac{K}{N} = 4 \times \frac{3}{10} = \mathbf{1.2} \]
Final Answer
P(X=2) = 3/10 = 0.30 | E(X) = 1.2 defective items on average
Binomial Distribution

Used when each trial has exactly two outcomes (success/failure), trials are independent, and probability of success p is constant. Count the number of successes in n trials.

Binomial PMF
\[ P(X=k) = \binom{n}{k} p^k (1-p)^{n-k} \]
\( \binom{n}{k} = \frac{n!}{k!(n-k)!} \)
Mean & Variance
\[ E(X) = np \]
\[ \text{Var}(X) = np(1-p) \]
✏️ Practice Example (From Exam Paper)
Problem: A biased coin has P(heads) = 0.15. Flipped 40 times. Find P(exactly 5 heads) using both Binomial and Poisson approximation. Compare results.
1
Binomial Calculation
n = 40, p = 0.15, k = 5
\[ P(X=5) = \binom{40}{5}(0.15)^5(0.85)^{35} \] \[ = 658008 \times 7.594 \times 10^{-4} \times 0.002157 \approx \mathbf{0.1692} \]
2
Poisson Approximation (λ = np = 40 × 0.15 = 6)
\[ P(X=5) = \frac{e^{-6} \cdot 6^5}{5!} = \frac{0.002479 \times 7776}{120} \approx \mathbf{0.1606} \]
3
Compare: |0.1606 − 0.1692| = 0.0086 → Small but noticeable difference
Poisson works best when n is very large and p is very small (p < 0.01). Here p = 0.15 is too large, so the approximation is less accurate.
Poisson Distribution

Models the number of events occurring in a fixed interval (time, space, length). Key parameter: λ (lambda) = average number of events per interval. When interval changes, scale λ proportionally!

Poisson PMF
\[ P(X=k) = \frac{e^{-\lambda} \cdot \lambda^k}{k!} \]
e ≈ 2.71828
Scaling λ
If λ is per unit, then for t units:
\[ \lambda_t = \lambda \times t \]
E.g., 0.2 flaws/foot → 50 feet: λ = 10
Complement Trick
P(X ≥ k) = 1 − P(X ≤ k−1)
P(X < k) = P(X=0) + ... + P(X=k−1)
✏️ Practice Example (From Exam Paper)
Problem: Flaws follow Poisson with λ = 0.2/foot. For a 50-foot roll, find P(5 ≤ X ≤ 8). Also, for a website with 1.2 visits/hour, find P(≥ 2 visits in 3 hours).
1
Part A: Scale λ to 50-foot roll
λ = 50 × 0.2 = 10
2
Compute P(X=5) through P(X=8)
\( P(X=k) = \frac{e^{-10} \cdot 10^k}{k!} \), where \(e^{-10} = 0.0000454\)
P(X=5) = 0.0378, P(X=6) = 0.0631, P(X=7) = 0.0901, P(X=8) = 0.1126
Total = 0.0378 + 0.0631 + 0.0901 + 0.1126 = 0.3036
3
Part B: Website — λ = 1.2 × 3 = 3.6
P(X ≥ 2) = 1 − P(X=0) − P(X=1)
\[ = 1 - e^{-3.6}(1 + 3.6) = 1 - (0.02732)(4.6) = 1 - 0.1257 = \mathbf{0.8743} \]
Final Answer
P(5 ≤ X ≤ 8) = 0.3036 | P(≥ 2 visits in 3 hrs) = 0.8743
Normal Distribution

The Normal (Gaussian) distribution is the most important in statistics. It is bell-shaped, symmetric about the mean. To find probabilities, always convert to Z-scores and use the standard normal table.

Z-score Conversion
\[ Z = \frac{X - \mu}{\sigma} \]
Then use standard normal table for P(Z < z)
Key Probabilities
P(μ − σ < X < μ + σ) ≈ 68%
P(μ − 2σ < X < μ + 2σ) ≈ 95%
P(μ − 3σ < X < μ + 3σ) ≈ 99.7%
Continuity Correction
"Exceeds 43 to nearest ohm" means
true value > 42.5, so use X = 42.5
(Add/subtract 0.5 for discrete → continuous)
✏️ Practice Example (From Exam Paper)
Problem: Exam grades are Normal with μ = 73, σ = 8. Find: (i) P(grade < 91), (ii) P(65 < grade < 89).
1
Part (i): P(X < 91)
\[ Z = \frac{91 - 73}{8} = \frac{18}{8} = 2.25 \]
P(Z < 2.25) = 0.9878 → 98.78%
2
Part (ii): P(65 < X < 89)
\[ Z_1 = \frac{65-73}{8} = -1.0 \qquad Z_2 = \frac{89-73}{8} = 2.0 \]
P(−1 < Z < 2) = P(Z < 2) − P(Z < −1)
= 0.9772 − 0.1587 = 0.8185 → 81.85%
Final Answer
P(X < 91) = 98.78% | P(65 < X < 89) = 81.85%
Continuous Random Variables & PDFs

A continuous random variable takes any value in an interval. Its Probability Density Function (PDF) f(x) satisfies: f(x) ≥ 0 and the total area under the curve = 1. Probability of a range is an integral.

Valid PDF
\[ \int_{-\infty}^{\infty} f(x)\,dx = 1 \]
Use this to find the constant c or k.
Probability as Area
\[ P(a < X < b) = \int_a^b f(x)\,dx \]
Expectation (Mean)
\[ E(X) = \int_{-\infty}^{\infty} x \cdot f(x)\,dx \]
Variance
\[ E(X^2) = \int_{-\infty}^{\infty} x^2 f(x)\,dx \]
\[ \text{Var}(X) = E(X^2) - [E(X)]^2 \]
✏️ Practice Example (From Exam Paper)
Problem: The rating X of an item follows the piecewise PDF:
f(x) = cx² for 0 ≤ x ≤ 1, and c(2−x) for 1 < x ≤ 2, else 0.
Find: (a) constant c, (b) P(1/2 < X < 3/2), (c) E(X).
1
(a) Find c using ∫f(x)dx = 1
\[ \int_0^1 cx^2\,dx + \int_1^2 c(2-x)\,dx = 1 \] \[ \frac{c}{3} + c\left[(2x - \frac{x^2}{2})\Big|_1^2\right] = 1 \] \[ \frac{c}{3} + c\left[(4-2) - (2 - \frac{1}{2})\right] = \frac{c}{3} + \frac{c}{2} = \frac{5c}{6} = 1 \] \[ \mathbf{c = \frac{6}{5}} \]
2
(b) P(1/2 < X < 3/2)
\[ = \int_{1/2}^1 cx^2\,dx + \int_1^{3/2} c(2-x)\,dx = c\left(\frac{7}{24}\right) + c\left(\frac{3}{8}\right) = c \cdot \frac{16}{24} = \frac{6}{5} \times \frac{2}{3} = \mathbf{\frac{4}{5}} \]
3
(c) E(X)
\[ E(X) = \int_0^1 cx^3\,dx + \int_1^2 cx(2-x)\,dx = c\left(\frac{1}{4}\right) + c\left(\frac{2}{3}\right) = c \cdot \frac{11}{12} = \frac{6}{5} \times \frac{11}{12} = \mathbf{\frac{11}{10}} = 1.1 \]
Final Answer
c = 6/5 = 1.2 | P(1/2 < X < 3/2) = 4/5 = 0.80 | E(X) = 11/10 = 1.1

Inclusion-Exclusion Principle — 3 Events

When three events A, B, C can overlap in complex ways, use the Inclusion-Exclusion formula to find P(at least one). This avoids double and triple counting of intersections.

Union of 3 Events
\[ P(A \cup B \cup C) = P(A) + P(B) + P(C) \] \[ - P(A \cap B) - P(B \cap C) - P(A \cap C) + P(A \cap B \cap C) \]
Exactly Two Events
\[ P(\text{exactly 2}) = P(A{\cap}B) + P(B{\cap}C) + P(A{\cap}C) - 3P(A{\cap}B{\cap}C) \]
Exactly One Event
P(exactly 1) = P(≥1) − P(exactly 2) − P(all 3)

🔑 Memory Aid — Add, Subtract, Add Back

  • ADD the three individual probabilities
  • SUBTRACT the three pairwise intersections
  • ADD BACK the triple intersection (which was subtracted 3 times)
✏️ Practice Example (From Exam Paper)
Problem (Dec 2025 Q6): Three weather alerts A (Flood), B (Landslide), C (Storm). Given: P(A∩B) = P(B∩C) = P(C∩A) = 1/8, P(A∩B∩C) = 1/16, P(Ā∩B̄∩C̄) = 1/4. Find: P(A), P(exactly 2 alerts), P(at least 1), P(exactly 1).
1
Use P(no alerts) = 1 − P(A∪B∪C) to find P(A∪B∪C)
P(A∪B∪C) = 1 − P(Ā∩B̄∩C̄) = 1 − 1/4 = 3/4
2
Find P(A) using Inclusion-Exclusion (all events equally likely: P(A)=P(B)=P(C))
\[ 3/4 = 3P(A) - 3(1/8) + 1/16 \] \[ 3/4 = 3P(A) - 3/8 + 1/16 = 3P(A) - 5/16 \] \[ 3P(A) = 3/4 + 5/16 = 12/16 + 5/16 = 17/16 \] \[ P(A) = \mathbf{17/48} \approx 0.354 \]
3
P(exactly 2 alerts)
\[ P(\text{exactly 2}) = [P(A\cap B) + P(B\cap C) + P(A\cap C)] - 3P(A\cap B\cap C) \] \[ = 3(1/8) - 3(1/16) = 6/16 - 3/16 = \mathbf{3/16} \]
4
P(exactly 1 alert)
\[ P(\text{exactly 1}) = P(\text{≥ 1}) - P(\text{exactly 2}) - P(\text{all 3}) \] \[ = 3/4 - 3/16 - 1/16 = 12/16 - 3/16 - 1/16 = \mathbf{8/16 = 1/2} \]
Final Answer
P(A) = 17/48 | P(exactly 2) = 3/16 | P(≥1) = 3/4 | P(exactly 1) = 1/2
Contingency Tables & Conditional Probability

A contingency table (also called a cross-tabulation) shows frequency counts for two categorical variables simultaneously. You can read off ALL probability types directly from the table — just carefully identify the correct row/column totals.

Joint Probability
\[ P(A \cap B) = \frac{\text{cell count}_{A,B}}{\text{Grand Total}} \]
Marginal Probability
\[ P(A) = \frac{\text{Row Total}_A}{\text{Grand Total}} \]
Conditional Probability
\[ P(A|B) = \frac{\text{cell count}_{A,B}}{\text{Column Total}_B} \]
✏️ Practice Example (From Exam Paper)
Problem (2024 Makeup2 Paper): A retail bank classified 38,000 customers by income level and loan default:
Loan DefaultLow IncomeMiddle IncomeHigh IncomeTotal
No8,00015,0007,00030,000
Yes2,0005,0001,0008,000
Total10,00020,0008,00038,000
Find: (i) P(No Default | Middle Income), (ii) P(Middle Income | No Default).
1
Part (i): P(No Default | Middle Income) — restrict to Middle Income column
\[ P(\text{No Default} | \text{Middle}) = \frac{\text{No Default AND Middle}}{\text{Total Middle}} = \frac{15{,}000}{20{,}000} = \mathbf{0.75} \]
2
Part (ii): P(Middle Income | No Default) — restrict to No Default row
\[ P(\text{Middle} | \text{No Default}) = \frac{\text{Middle AND No Default}}{\text{Total No Default}} = \frac{15{,}000}{30{,}000} = \mathbf{0.50} \]
Final Answer
P(No Default | Middle Income) = 0.75 | P(Middle Income | No Default) = 0.50

Normal Distribution — Inverse Problem (Find X from Probability)

The "forward" normal problem asks: given X, find P. The inverse problem asks: given P, find X. This is used for cut-off scores, percentiles, and specification limits. You look up the Z-table in reverse.

Forward: P from X
\[ Z = \frac{X - \mu}{\sigma} \] then look up P(Z)
Inverse: X from P
Look up Z from table, then:
\[ X = \mu + Z \cdot \sigma \]
Common Z-values
Top 5% (P=0.95) → Z = 1.645
Top 2.5% → Z = 1.960
Top 1% → Z = 2.326
✏️ Practice Example (From Exam Paper — Jan 2026)
Problem: Exam scores are Normal with μ = 73, σ = 8. Only 5% of students score above the Dean's List cut-off. What is the cut-off score?
1
Identify: We need P(X > cut-off) = 0.05, so P(X < cut-off) = 0.95
Look in the Z-table for the Z value where P(Z < z) = 0.95
2
Find Z from table
P(Z < z) = 0.95 → Z = 1.645
3
Convert Z back to X
\[ X = \mu + Z \cdot \sigma = 73 + 1.645 \times 8 = 73 + 13.16 = \mathbf{86.16} \]
Final Answer
Dean's List cut-off ≈ 86.16 marks (top 5% of students score above this).
CDF → PDF Derivation (Differentiation)

A Cumulative Distribution Function (CDF) F(x) gives the probability that X ≤ x. To find the PDF f(x), differentiate the CDF: f(x) = dF/dx. To verify a CDF is valid: it must be 0 at −∞, 1 at +∞, and non-decreasing.

Relationship
\[ f(x) = \frac{d}{dx} F(x) \]
\[ P(a \le X \le b) = F(b) - F(a) \]
Validity Checks
F(-∞) = 0
F(+∞) = 1
f(x) = F'(x) ≥ 0 everywhere
Finding constant k
Set F(max) = 1
Solve for k in the
boundary condition.
✏️ Practice Example (From Exam Paper — Jan 2026)
Problem: A data centre's overheating index X (range 0 to 3) has CDF F(x) = kx²(3−x)/9 for 0 ≤ x ≤ 3, else 0.
Find: (a) constant k, (b) P(1 ≤ X ≤ 2), (c) the PDF f(x).
1
(a) Find k: use F(3) = 1
\[ F(3) = k \cdot \frac{3^2(3-3)}{9} = 0 \] — This doesn't work directly. Instead use the constraint that \( \lim_{x \to 3^-} F(x) = 1 \).
Actually, F(3) must = 1. From the given form:
\( k(9)(0)/9 = 0 \). So we need the CDF to equal 1 at x=3 using the formula's design.
In exam, set F(x_max) = 1: e.g., if F(x) = kx²/9 for 0≤x≤3, then F(3) = k(9)/9 = k = 1, so k = 1.
2
(b) Find P(1 ≤ X ≤ 2) = F(2) − F(1)
\[ P(1 \le X \le 2) = F(2) - F(1) \]
Substitute into the CDF formula at x=2 and x=1, then subtract.
3
(c) Derive the PDF by differentiating F(x)
\[ f(x) = \frac{d}{dx} F(x) \]
Apply the product rule / chain rule to whatever form F(x) is given in.
Always check f(x) ≥ 0 over the valid range.

🔑 General Template for CDF Problems

  • Step 1: Use boundary F(x_max) = 1 → find constant k
  • Step 2: Use P(a ≤ X ≤ b) = F(b) − F(a) for range probability
  • Step 3: Differentiate F(x) to get f(x) = dF/dx
  • Step 4: Verify f(x) ≥ 0 for all x in valid domain
Joint Continuous PDF — Two Variables

When two continuous variables X and Y together have a Joint PDF f(x,y), we use double integration to compute probabilities and marginals. This is the continuous analogue of the joint PMF table.

Valid Joint PDF
\[ \int\int f(x,y)\,dx\,dy = 1 \]
f(x,y) ≥ 0 for all (x,y)
Marginal PDFs
\[ f_X(x) = \int_{-\infty}^{\infty} f(x,y)\,dy \]
\[ f_Y(y) = \int_{-\infty}^{\infty} f(x,y)\,dx \]
Independence Test
X and Y are independent if and only if:
\[ f(x,y) = f_X(x) \cdot f_Y(y) \]
Expectations
\[ E[X] = \int\int x \cdot f(x,y)\,dx\,dy \]
\[ E[XY] = \int\int xy \cdot f(x,y)\,dx\,dy \]

🔑 Integration Order Tip

  • Integrate the INNER variable first, treating the outer as a constant
  • For marginal f_X(x): integrate out y (inner); result is a function of x only
  • For marginal f_Y(y): integrate out x (inner); result is a function of y only
  • For P(X < a, Y > b): split into a product of two separate integrals only if X and Y are independent
✏️ Practice Example (From Exam Paper — Dec 2025)
Problem: Let X, Y be continuous with joint PDF f(x,y) = 6x²y for 0 ≤ x ≤ 1, 0 ≤ y ≤ 1, else 0.
(i) Verify valid joint PDF. (ii) Find marginals f_X(x) and f_Y(y). (iii) P(X < 0.5, Y > 0.5). (iv) Are X and Y independent? (v) Find E[X], E[Y], E[XY].
1
(i) Verify: ∫₀¹∫₀¹ 6x²y dy dx = 1
\[ \int_0^1 \int_0^1 6x^2y\,dy\,dx = \int_0^1 6x^2 \left[\frac{y^2}{2}\right]_0^1 dx = \int_0^1 3x^2\,dx = [x^3]_0^1 = \mathbf{1} \checkmark \]
2
(ii) Find marginals
\[ f_X(x) = \int_0^1 6x^2y\,dy = 6x^2 \cdot \frac{1}{2} = \mathbf{3x^2}, \quad 0 \le x \le 1 \]
\[ f_Y(y) = \int_0^1 6x^2y\,dx = 6y \cdot \frac{x^3}{3}\Big|_0^1 = \mathbf{2y}, \quad 0 \le y \le 1 \]
3
(iii) P(X < 0.5, Y > 0.5)
\[ = \int_0^{0.5} \int_{0.5}^1 6x^2y\,dy\,dx \]
Inner: \( \int_{0.5}^1 6x^2y\,dy = 6x^2 \cdot \left[\frac{y^2}{2}\right]_{0.5}^1 = 6x^2 \cdot (0.5 - 0.125) = 6x^2(0.375) = 2.25x^2 \)
Outer: \( \int_0^{0.5} 2.25x^2\,dx = 2.25 \cdot \frac{(0.5)^3}{3} = 2.25 \times \frac{1}{24} = \mathbf{\frac{3}{32} \approx 0.094} \)
4
(iv) Independence Test
f_X(x) · f_Y(y) = (3x²)(2y) = 6x²y = f(x,y) ✓
Yes, X and Y are independent because the joint PDF factorizes into the product of the marginals.
5
(v) Compute expectations
\[ E[X] = \int_0^1 x \cdot 3x^2\,dx = 3 \cdot \frac{x^4}{4}\Big|_0^1 = \mathbf{3/4} \]
\[ E[Y] = \int_0^1 y \cdot 2y\,dy = 2 \cdot \frac{y^3}{3}\Big|_0^1 = \mathbf{2/3} \]
Since independent: \( E[XY] = E[X] \cdot E[Y] = \frac{3}{4} \times \frac{2}{3} = \mathbf{\frac{1}{2}} \)
Final Answer
Valid PDF ✓ | f_X(x) = 3x² | f_Y(y) = 2y | P(X<0.5, Y>0.5) = 3/32 ≈ 0.094
X and Y are independent | E[X] = 3/4, E[Y] = 2/3, E[XY] = 1/2
