📊 Descriptive Statistics

Mean & Variance

Sample Mean\(\bar{x} = \dfrac{\sum x_i}{n}\)
Sample Variance\(s^2 = \dfrac{\sum(x_i - \bar{x})^2}{n-1}\)
Std Deviation\(s = \sqrt{s^2}\)
Population Var\(\sigma^2 = \dfrac{\sum(x_i - \mu)^2}{N}\)
⚠ Use n−1 (sample) vs N (population)
📦

Quartiles & IQR

Step 1Sort data ascending
Q1 position\(\dfrac{n+1}{4}\)-th value
Q2 (Median)\(\dfrac{n+1}{2}\)-th value
Q3 position\(\dfrac{3(n+1)}{4}\)-th value
IQR\(IQR = Q3 - Q1\)
Outlier boundsLower: \(Q1 - 1.5 \cdot IQR\)  |  Upper: \(Q3 + 1.5 \cdot IQR\)
🎲 Probability Rules
🎲

Core Probability

Addition Rule\(P(A \cup B) = P(A)+P(B)-P(A \cap B)\)
Complement\(P(A') = 1 - P(A)\)
Conditional\(P(A|B) = \dfrac{P(A \cap B)}{P(B)}\)
Multiplication\(P(A \cap B) = P(A|B) \cdot P(B)\)
Independence\(P(A \cap B) = P(A) \cdot P(B)\)
Set difference\(P(A-B) = P(A) - P(A \cap B)\)
Union (sets)\(P(A \cup B) = P(A{-}B)+P(B{-}A)+P(A{\cap}B)\)
🧠

Bayes' Theorem

$$P(A|B) = \frac{P(B|A)\,P(A)}{P(B)}$$
Law of Total Prob\(P(B) = \sum_i P(B|A_i)\,P(A_i)\)
Full form\(P(A_i|B) = \dfrac{P(B|A_i)\,P(A_i)}{\sum_j P(B|A_j)\,P(A_j)}\)
💡 Prior × Likelihood ÷ Evidence
📈 Probability Distributions
🎯

Discrete RV

Valid PMF\(\sum_x P(X=x) = 1\)  and  \(P(x) \geq 0\)
Expected Value\(E(X) = \sum x \cdot P(x)\)
E(X²)\(E(X^2) = \sum x^2 \cdot P(x)\)
Variance\(Var(X) = E(X^2) - [E(X)]^2\)
E(aX+b)\(aE(X) + b\)
Var(aX+b)\(a^2 \cdot Var(X)\)  (b vanishes!)
🔵

Binomial Distribution

$$P(X=k) = \binom{n}{k} p^k (1-p)^{n-k}$$
Mean\(\mu = np\)
Variance\(\sigma^2 = np(1-p)\)
Combination\(\binom{n}{k} = \dfrac{n!}{k!(n-k)!}\)
Use whenFixed n, independent trials, constant p
🟢

Poisson Distribution

$$P(X=k) = \frac{e^{-\lambda}\,\lambda^k}{k!}$$
Mean = Variance\(E(X) = Var(X) = \lambda\)
Approx to Binom\(\lambda = np\)  (large n, small p)
Scaling intervalFor interval t: \(\lambda_t = \lambda \cdot t\)
P(≥k)\(1 - \sum_{x=0}^{k-1} \frac{e^{-\lambda}\lambda^x}{x!}\)
🟠

Hypergeometric

$$P(X=k) = \frac{\binom{K}{k}\binom{N-K}{n-k}}{\binom{N}{n}}$$
NTotal population size
KTotal successes in population
nSample size drawn (without replacement)
⚡ Without replacement → always Hypergeometric
📉 Continuous Random Variables
🌊

Continuous RV (PDF / CDF)

Valid PDF\(\int_{-\infty}^{\infty} f(x)\,dx = 1\)  and  \(f(x)\geq 0\)
Probability\(P(a < X < b) = \int_a^b f(x)\,dx\)
Expected Value\(E(X) = \int_{-\infty}^{\infty} x\,f(x)\,dx\)
E(X²)\(\int_{-\infty}^{\infty} x^2\,f(x)\,dx\)
Variance\(Var(X) = E(X^2) - [E(X)]^2\)
💡 Find k or c: set integral = 1 and solve
🔔

Normal Distribution

$$Z = \frac{X - \mu}{\sigma}$$
StandardiseConvert X to Z, look up Z-table
P(X < a)\(\Phi\!\left(\dfrac{a-\mu}{\sigma}\right)\)
P(a<X<b)\(\Phi(Z_b) - \Phi(Z_a)\)
P(X > a)\(1 - \Phi(Z_a)\)
CLT / Sampling\(Z = \dfrac{\bar{X}-\mu}{\sigma/\sqrt{n}}\)
⚡ \(\Phi(-z) = 1 - \Phi(z)\) — use symmetry!
🔗 Joint Distributions
🔗

Joint, Marginal & Conditional

Marginal P(X)\(P_X(x) = \sum_y P(x,y)\)
Marginal P(Y)\(P_Y(y) = \sum_x P(x,y)\)
Conditional X|Y\(P(X=x\mid Y=y) = \dfrac{P(x,y)}{P_Y(y)}\)
Conditional Y|X\(P(Y=y\mid X=x) = \dfrac{P(x,y)}{P_X(x)}\)
Independence\(P(x,y) = P_X(x)\cdot P_Y(y)\) for all x,y
Joint sum checkAll cells in joint table must sum to 1
🤖 Naive Bayes Classifier
🤖

Naive Bayes Classifier

$$P(C\mid\text{doc}) \propto P(C) \prod_{i} P(w_i\mid C)$$
Prior\(P(C) = \dfrac{\text{docs in class }C}{\text{total docs}}\)
Likelihood\(P(w\mid C) = \dfrac{\text{count}(w,C)}{\text{total words in }C}\)
Laplace smooth\(P(w\mid C) = \dfrac{\text{count}(w,C)+1}{\text{total words in }C + |V|}\)
⚡ |V| = total unique words in vocabulary
📋 Step-by-step
① Count words per class & total words
② Compute prior P(C)
③ Compute P(w|C) with Laplace smoothing
④ Multiply prior × all word likelihoods per class
⑤ Pick class with higher result = classification
📏 Confidence Intervals & Sample Size
📏

Confidence Intervals

CI for μ (known σ)\(\bar{x} \pm z_{\alpha/2}\dfrac{\sigma}{\sqrt{n}}\)
CI for μ (unknown σ)\(\bar{x} \pm t_{\alpha/2,\,n-1}\dfrac{s}{\sqrt{n}}\)
Margin of Error\(E = z_{\alpha/2}\dfrac{\sigma}{\sqrt{n}}\)
Required Sample n\(n = \left(\dfrac{z_{\alpha/2}\cdot\sigma}{E}\right)^2\)  → Round UP
z-values90%→1.645  |  95%→1.96  |  99%→2.576
