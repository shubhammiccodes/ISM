MODULE 1: Basic Probability & Statistics

🎓 CS1: Central Tendency & Variability

Descriptive Statistics summarizes data. Central tendency refers to the "middle" of the data (Mean, Median, Mode), while variability measures how spread out the data is (Variance, Standard Deviation, IQR).

Five-Number Summary: Min, Q1 (25th percentile), Median (50th), Q3 (75th), Max. Useful for Box Plots.

Outlier Detection: Any value outside the interval [Q1 - 1.5(IQR), Q3 + 1.5(IQR)] is considered an outlier.

IQR = Q3 - Q1
Lower Fence = Q1 - 1.5 × IQR
Upper Fence = Q3 + 1.5 × IQR

🎓 CS2: Axioms of Probability

Sample Space (S): The set of all possible outcomes.
Mutually Exclusive: Two events cannot happen at the same time. \(P(A \cap B) = 0\).
Independent: The outcome of one event does not affect the other. \(P(A \cap B) = P(A)P(B)\).

Important Theorem Proof: "Mutually exclusive events are independent." This is FALSE.
Proof: If A and B are mutually exclusive, \(P(A \cap B) = 0\). If they were also independent, then \(P(A \cap B) = P(A)P(B)\). This would mean \(P(A)P(B) = 0\), so either P(A) or P(B) must be 0. Thus, non-zero mutually exclusive events CANNOT be independent.

MODULE 2: Conditional Probability & Bayes

🎓 CS3: Conditional Probability

The probability of event A given that event B has already occurred.

\[ P(A|B) = \frac{P(A \cap B)}{P(B)} \] \[ \text{Law of Total Prob: } P(B) = \sum_{i} P(B|A_i)P(A_i) \]

📝 Worked Example: Road Accident Probabilities

Problem: P(A)=rash driving=15%, P(B)=bad roads=25%, P(C)=road indiscipline=20%, P(D)=other=40%.
Death rates: P(S|A)=1%, P(S|B)=2%, P(S|C)=3%, P(S|D)=4%. Find overall death rate P(S).

Solution (Law of Total Probability):
P(S) = P(A)P(S|A) + P(B)P(S|B) + P(C)P(S|C) + P(D)P(S|D)
P(S) = (0.15 × 0.01) + (0.25 × 0.02) + (0.20 × 0.03) + (0.40 × 0.04)
P(S) = 0.0015 + 0.0050 + 0.0060 + 0.0160 = 0.0285 (2.85%)

📝 Worked Example: Probability Algebra

Given: P(A)=1/2, P(B)=1/3, P(A|B)=1/6. Find P(B|A) and P(B|A').

Step 1: Intersection
P(A ∩ B) = P(A|B) × P(B) = (1/6) × (1/3) = 1/18

Step 2: P(B|A)
P(B|A) = P(A ∩ B) / P(A) = (1/18) / (1/2) = 1/9

Step 3: P(B|A')
P(A' ∩ B) = P(B) - P(A ∩ B) = 1/3 - 1/18 = 6/18 - 1/18 = 5/18
P(A') = 1 - P(A) = 1 - 1/2 = 1/2
P(B|A') = P(A' ∩ B) / P(A') = (5/18) / (1/2) = 5/9

🎓 CS4: Bayes' Theorem & Naive Bayes

Bayes' Theorem reverses conditional probabilities. It computes the posterior probability based on prior probabilities and likelihoods.

Naive Bayes Classifier is widely used in text classification (like Spam detection). It assumes features (words) are conditionally independent given the class.

Laplace Smoothing: Used to prevent zero probabilities for unseen words. Add \(\alpha\) (usually 1) to the numerator, and add \(\alpha \times |V|\) to the denominator, where |V| is the total vocabulary size.

MODULE 3: Probability Distributions

🎓 CS5: Random Variables

Discrete RV: Takes specific, distinct values. Modeled by Probability Mass Function (PMF). \(\sum P(X=x) = 1\).

Continuous RV: Takes any value in a range. Modeled by Probability Density Function (PDF). \(\int f(x)dx = 1\).

Expectation (Mean): \( E[X] = \sum x P(x) \) OR \( \int x f(x)dx \)
Variance: \( Var(X) = E[X^2] - (E[X])^2 \)

🎓 CS6: Named Distributions

  • Binomial: Number of successes in n independent trials. \( P(X=k) = \binom{n}{k} p^k (1-p)^{n-k} \). Mean = np.
  • Poisson: Number of events in fixed interval. \( P(X=k) = \frac{e^{-\lambda} \lambda^k}{k!} \). Mean = \(\lambda\). Used to approximate Binomial when n is large and p is small (\(\lambda = np\)).
  • Normal: Bell curve. Characterized by mean \(\mu\) and SD \(\sigma\). Standardize to Z-score: \( Z = \frac{X - \mu}{\sigma} \).
