⚡ Quick Formula Reference Card

Mean
x̄ = (Σxᵢ) / n
Sample Standard Deviation
s = √[Σ(xᵢ-x̄)² / (n-1)]
IQR
IQR = Q3 - Q1
Outlier Fences
Lower: Q1 - 1.5×IQR | Upper: Q3 + 1.5×IQR
Conditional Probability
P(A|B) = P(A∩B) / P(B)
Bayes' Theorem
P(A|B) = P(B|A)P(A) / P(B)
Total Probability
P(B) = ΣP(B|Aᵢ)P(Aᵢ)
Inclusion-Exclusion
P(A∪B) = P(A)+P(B)-P(A∩B)
Binomial PMF
P(X=k) = C(n,k) × pᵏ × (1-p)ⁿ⁻ᵏ
Poisson PMF
P(X=k) = e⁻λ × λᵏ / k!
Z-Score (Standardization)
z = (x - μ) / σ
Expectation (Discrete)
E[X] = Σ xᵢ × P(X=xᵢ)
Variance
Var(X) = E[X²] - (E[X])²
Standard Error (CLT)
SE = σ / √n
Sample Size Formula
n = (z × σ / E)²
Valid PDF Condition
∫f(x)dx = 1, f(x) ≥ 0
Var(aX+b)
Var(aX+b) = a² × Var(X)
Poisson Approximation
λ = np (when n large, p small)

💡 Exam Preparation Tips from Your Professor

1

Always sort the data first before finding quartiles, median, or any order statistics. This prevents mistakes.

2

For Bayes' Theorem, always write out all given probabilities and identify what you're looking for before computing.

3

For Naive Bayes classifier, always apply Laplace smoothing (add 1 to numerator, add |V| to denominator).

4

Use Poisson approximation to Binomial only when n is large (≥20) and p is small (≤0.05). λ = np.

5

For continuous PDFs, always verify using ∫f(x)dx = 1 to find the constant before computing probabilities.

6

For joint distributions, check independence by verifying f(x,y) = fX(x) × fY(y). Don't guess!

7

Normal distribution: always convert to Z-scores first, then use the standard normal table.

8

For PMF problems, use ΣP(X=x) = 1 to find constants k. Never skip this validation step.
