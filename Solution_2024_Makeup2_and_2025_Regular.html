Q1. Descriptive Stats & Set Theory
[6 Marks]
a) Weights: 30, 32, 35, 38, 40, 42, 45, 48, 50, 53. Calculate Q1, Q2, Q3, IQR, and determine outliers.

b) Set of workers. M: Machine operators, Q: Quality inspectors.
P(M∪Q)=0.40, P(M-Q)=0.15, P(Q-M)=0.10. Find P(M∩Q), P(M), P(Q).

📝 Solution

Part a) Quartiles & Outliers
Data (sorted): 30, 32, 35, 38, 40, 42, 45, 48, 50, 53. (n=10)
Q1 position = 0.25(11) = 2.75. Q1 = 32 + 0.75(35-32) = 34.25
Median (Q2) = average of 5th and 6th = (40+42)/2 = 41
Q3 position = 0.75(11) = 8.25. Q3 = 48 + 0.25(50-48) = 48.5
IQR = Q3 - Q1 = 48.5 - 34.25 = 14.25
Lower fence = 34.25 - 1.5(14.25) = 12.875
Upper fence = 48.5 + 1.5(14.25) = 69.875
Conclusion: No outliers present.
Part b) Set Theory
We know: \( P(M \cup Q) = P(M-Q) + P(Q-M) + P(M \cap Q) \)
0.40 = 0.15 + 0.10 + P(M ∩ Q)
P(M ∩ Q) = 0.40 - 0.25 = 0.15

P(M) = P(M-Q) + P(M ∩ Q) = 0.15 + 0.15 = 0.30
P(Q) = P(Q-M) + P(M ∩ Q) = 0.10 + 0.15 = 0.25
Q2. Normal Distribution & Total Probability
a) Normal dist: μ=73, σ=8. Find P(X < 91) and P(65 < X < 89).

b) Ram Nagar: P(Open)=0.25, P(Closed)=0.75. P(Stolen|Open)=0.05, P(Stolen|Closed)=0.01. Find P(Open|Stolen).

📝 Solution

Part a) Normal Distribution
i) \( z = \frac{91-73}{8} = 2.25 \). P(Z < 2.25) = 0.9878 = 98.78%
ii) \( z_1 = \frac{65-73}{8} = -1.0 \), \( z_2 = \frac{89-73}{8} = 2.0 \)
P(-1.0 < Z < 2.0) = P(Z < 2.0) - P(Z < -1.0) = 0.9772 - 0.1587 = 0.8185 (81.85%)
Part b) Bayes' Theorem
P(Stolen) = P(Stolen|Open)×P(Open) + P(Stolen|Closed)×P(Closed)
= 0.05×0.25 + 0.01×0.75 = 0.0125 + 0.0075 = 0.02
P(Open|Stolen) = \(\frac{0.05 \times 0.25}{0.02}\) = \(\frac{0.0125}{0.02}\) = 0.625 (62.5%)
Q3. Contingency Tables & Poisson/Binomial

📝 Solution Highlight

Part b) Comparing Binomial & Poisson
n=25, p=0.2. Find P(X=3).
Binomial: \( P(X=3) = \binom{25}{3} 0.2^3 (0.8)^{22} = 2300 \times 0.008 \times 0.007374 = \mathbf{0.1358} \)
Poisson: \(\lambda = np = 5\). \( P(X=3) = \frac{e^{-5} 5^3}{3!} = \mathbf{0.1404} \)
Conclusion: Poisson slightly overestimates since p=0.2 is not very small.
Q5. Joint PMF

📝 Solution Highlight

Given a 4x4 matrix of joint probabilities. To find Marginal PMF, simply sum across the rows (for X) or down the columns (for Y).
\( E(X) = \sum x \cdot P(X=x) \)
\( Var(X) = E(X^2) - (E(X))^2 \)

Q3. Spam Filter via Naive Bayes

📝 Solution Highlight

Priors: P(Spam) = 0.5, P(Not Spam) = 0.5. Vocabulary |V| = 25.
With Laplace Smoothing (\(\alpha=1\)):
P(word | Spam) = \(\frac{\text{count in Spam} + 1}{\text{total Spam words} + |V|}\) = \(\frac{\text{count} + 1}{14 + 25} = \frac{\text{count} + 1}{39}\)
P(word | NotSpam) = \(\frac{\text{count in NotSpam} + 1}{11 + 25} = \frac{\text{count} + 1}{36}\)
Comparing the multiplied likelihoods, the email is classified as SPAM.
Q6. Joint Distribution (Coin Toss Game)
Fair coin tossed 3 times. X = # of tails. Y = winnings ($3 if first tail on 1st toss, $2 if on 2nd, $1 if on 3rd, -$2 if no tail).

📝 Solution Highlight

Sample space (8 outcomes, each 1/8 prob):
HHH: X=0, Y=-2
HHT: X=1, Y=1
HTH: X=1, Y=2
HTT: X=2, Y=2
THH: X=1, Y=3
THT: X=2, Y=3
TTH: X=2, Y=3
TTT: X=3, Y=3

Marginal PMF of X: P(X=0)=1/8, P(X=1)=3/8, P(X=2)=3/8, P(X=3)=1/8.
\( E(X) = 0(1/8) + 1(3/8) + 2(3/8) + 3(1/8) = 1.5 \)
